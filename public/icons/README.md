# PWA Icons

This directory contains icons for the Progressive Web App.

## Required Icons

The following icon sizes are required:
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Generating Icons

### Option 1: Use the HTML Generator

1. Open `generate-icons.html` in your browser
2. Click "Generate Icons"
3. Icons will be downloaded automatically

### Option 2: Use an Online Tool

Use a tool like [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator) or [RealFaviconGenerator](https://realfavicongenerator.net/) to generate all required sizes from a single source image.

### Option 3: Manual Creation

Create icons with:
- Background: Gradient from #2d2d44 to #1e1e2e
- Icon: Lock emoji (üîê) or lock symbol in #6b8dd6
- Rounded corners: ~15% border radius
- Format: PNG
